现在愿意静下心读书的人也许没有以前多了，但书依然是了解世界成本最低的方式之一。只要你喜欢读书，或者突然想找一本书看看，互联网其实提供了许多免费的公共版权项目、开放获取平台和数字图书馆。

免费不等于无版权。 本文优先推荐公共版权、开放许可、作者或出版社授权，以及图书馆提供的数字借阅资源。一本原著进入公共领域，并不代表它的新译本、注释版、封面和排版也没有版权。

公共版权与经典文学

Project Gutenberg：最经典的免费电子书项目

  • 访问地址：https://www.gutenberg.org/
  • 是否注册：不需要
  • 常见格式：EPUB、Kindle、HTML、TXT

Project Gutenberg（古登堡计划）是历史最悠久的免费电子书项目之一，收录重点是版权已经到期的经典作品。莎士比亚、狄更斯、简·奥斯汀、马克·吐温和柯南·道尔等作家的英文原著都很容易找到。

网站可以按作者、书名、语言和主题检索，下载过程简单，没有强制注册。它尤其适合阅读英文经典和练习原版阅读，但中文藏书相对有限。

Standard Ebooks：排版更精致的英文公版书

  • 访问地址：https://standardebooks.org/
  • 是否注册：不需要
  • 常见格式：EPUB、AZW3、Kobo EPUB

Standard Ebooks 同样以公共版权英文作品为主，但它更加重视校对、封面、目录、字体和无障碍阅读。项目通常以 Project Gutenberg、Internet Archive 等平台的文本为基础，再由志愿者进行整理。

如果你使用 Kindle、Apple Books 或 Kobo 阅读经典文学，Standard Ebooks 的成品通常比未经整理的扫描版更舒服。

Wikisource：维基媒体旗下的自由图书馆

  • 中文站：https://zh.wikisource.org/
  • 多语言入口：https://wikisource.org/
  • 是否注册：阅读不需要

维基文库收录公共版权或采用自由许可发布的原始文本，内容包括文学作品、历史文献、法律文本、演讲、古籍和旧报刊。

它的优势是可以在线校对、全文检索和跨语言查找。部分作品支持导出为 EPUB、PDF 等格式，不过排版质量会因作品完成度而不同。

ManyBooks：界面更像现代电子书商店

  • 访问地址：https://manybooks.net/
  • 是否注册：部分功能可能需要注册
  • 常见格式：EPUB、PDF、在线阅读

ManyBooks 收录了大量经典作品和独立作者图书，分类、榜单和推荐功能比传统数字图书馆更接近现代网上书店。

需要注意的是，网站并非所有书籍都永久免费。下载前应查看价格和授权说明，不要把限时促销或试读内容误认为公共版权资源。

数字图书馆与在线借阅

Open Library：为每一本书建立一个页面

  • 访问地址：https://openlibrary.org/
  • 是否注册：搜索不需要，借阅通常需要
  • 获取方式：在线阅读、限时借阅、部分下载

Open Library 是 Internet Archive 旗下的开放图书目录。它适合按书名、作者或 ISBN 查询不同版本，并为部分馆藏提供数字借阅。

页面出现 Read 或 Borrow 时，代表该版本可以在线阅读或限时借阅；没有相应按钮时，它可能只是一条书目记录。借阅资源应遵守平台期限，不等同于可以永久下载和传播。

Internet Archive：规模庞大的数字档案馆

  • 图书入口：https://archive.org/details/texts
  • 是否注册：浏览不需要，部分借阅需要
  • 获取方式：在线阅读、公共版权下载、受限借阅

Internet Archive 不只保存网页，还保存图书、期刊、音频、电影、软件和历史资料。其 Texts 分类中有大量数字化书籍，其中公共版权资料通常可以直接下载，其他内容可能只能受限阅读或借阅。

每个条目的可用格式和版权状态不同，下载前应查看页面的 Usage、Rights & Access 或相关说明。

Google Books：查绝版书和完整视图

  • 访问地址：https://books.google.com/
  • 是否注册：搜索通常不需要
  • 获取方式：完整阅读、部分预览、部分 PDF 下载

Google Books 并不是所有图书都能免费下载。公共版权作品可能提供 Full View 和 PDF 下载，仍受版权保护的书通常只能预览有限页数。

它很适合查找旧版书、绝版书、目录信息和书中的具体内容。找到图书后，应留意页面标注的是完整阅读、预览还是仅提供书目信息。

HathiTrust Digital Library：高校联合数字馆藏

  • 访问地址：https://www.hathitrust.org/
  • 是否注册：搜索不需要，部分功能限合作机构用户
  • 获取方式：公共版权全文、受版权资料检索

HathiTrust 由多家高校和研究机构共同建设，保存大量数字化图书和期刊。公共版权资料通常可以完整阅读，受版权保护的内容可能只能搜索关键词或查看有限页面。

它更适合寻找历史文献、旧期刊、早期出版物和学术资料。

开放获取学术书籍

DOAB：开放获取学术图书目录

  • 访问地址：https://www.doabooks.org/
  • 是否注册：不需要
  • 获取方式：链接到出版社或仓储平台下载

DOAB 全称 Directory of Open Access Books，是一个面向开放获取学术图书的检索目录。它收录来自大学出版社、研究机构和专业出版社的同行评审著作，可以按主题、出版社和语言筛选。

如果你正在写论文、查资料或系统学习一个专业领域，DOAB 通常比普通网盘资源更可靠，也更容易确认图书的出版信息和授权方式。

OAPEN Library：开放获取学术专著

  • 访问地址：https://library.oapen.org/
  • 是否注册：不需要
  • 常见格式：PDF、EPUB

OAPEN 专注于同行评审的开放获取图书，覆盖人文、社会科学、法律、语言、艺术、教育等领域。大部分条目会明确标注许可证、出版社、作者和 DOI。

DOAB 更像跨平台目录，OAPEN Library 则直接托管了不少图书文件，两者可以配合使用。

JSTOR Open Content：JSTOR 的开放内容入口

  • 访问地址：https://www.jstor.org/open/
  • 是否注册：开放内容通常不需要
  • 获取方式：在线阅读、部分 PDF 下载

JSTOR 除了订阅数据库，也提供开放获取期刊文章、研究报告、图书和公共版权内容。搜索结果标注 Open Access 或 Free 时通常可以免费阅读；其余内容仍可能需要学校、图书馆或个人订阅。

National Academies Press：科学与公共政策报告

  • 访问地址：https://nap.nationalacademies.org/
  • 是否注册：在线阅读通常不需要，下载方式以页面说明为准
  • 常见格式：PDF、在线阅读

美国国家科学院出版社出版科学、工程、医学和公共政策方面的图书与研究报告，许多作品可以免费在线阅读或下载 PDF。这里的内容专业性较强，但来源清晰，适合查找研究综述和政策资料。

MIT Press Direct：开放获取专著

  • 开放获取入口：https://direct.mit.edu/books/pages/open-access
  • 是否注册：开放图书通常不需要
  • 获取方式：PDF、EPUB、在线阅读，具体以书籍页面为准

MIT Press 将部分学术专著以开放获取方式发布，计算机、人工智能、设计、媒体、认知科学和社会科学领域的读者可以重点关注。

免费大学教材

OpenStax：经过同行评审的开放教材

  • 访问地址：https://openstax.org/
  • 是否注册：阅读和下载通常不需要
  • 常见格式：网页、PDF

OpenStax 是莱斯大学的非营利教育项目，提供高中和大学阶段的开放教材，覆盖数学、物理、化学、生物、经济学、心理学、商业和计算机等学科。

教材采用开放许可，适合学生、自学者和教师使用。不同版本可能有网页、PDF、低分辨率 PDF 或配套练习，应在教材页面选择需要的格式。

Open Textbook Library：跨学科开放教材目录

  • 访问地址：https://open.umn.edu/opentextbooks
  • 是否注册：不需要
  • 获取方式：PDF、EPUB、网页或出版社页面

Open Textbook Library 由明尼苏达大学的 Open Education Network 支持，收录作者和出版社授权免费使用、下载及改编的开放教材。

网站覆盖商业、计算机、工程、数学、医学、自然科学和人文社科等学科，并提供教师评价，挑选教材时很有参考价值。

LibreTexts：可以按章节学习的开放教材库

  • 访问地址：https://libretexts.org/
  • 是否注册：阅读不需要
  • 获取方式：网页、PDF、可编辑课程内容

LibreTexts 更像一个持续更新的在线教材平台，涵盖生物、化学、工程、数学、医学、社会科学和人文学科。它适合按章节查概念，也适合教师组合课程材料。

中文古籍与东方文献

书格：高清中文古籍与艺术资料

  • 访问地址：https://www.shuge.org/
  • 是否注册：浏览不需要
  • 常见格式：PDF、图片

书格是一个很有特色的中文古籍数字图书馆，收录公共领域的古籍善本、绘画、舆图、书法和历史文献。每部作品通常附有版本介绍、馆藏来源和文件信息。

它保留了古籍原有的版式和书影，适合喜欢中国历史、古典文学、传统艺术和古籍装帧的读者。扫描古籍文件往往较大，下载前可以先查看文件尺寸。

中国哲学书电子化计划：全文检索中国古籍

  • 中文入口：https://ctext.org/zh
  • 是否注册：阅读和检索不需要
  • 获取方式：在线阅读、原文检索、影印本对照

中国哲学书电子化计划收录先秦至近代的大量中文典籍，提供全文检索、原文与影印本对照、字词查询和相关研究工具。

它更偏向在线阅读与研究，不一定为每部作品提供现成的 EPUB 或 PDF，但查找古籍原文非常方便。

日本国立国会图书馆数字馆藏

  • 访问地址：https://dl.ndl.go.jp/
  • 是否注册：公开资料通常不需要，部分馆藏有限制
  • 获取方式：在线阅读、图片或 PDF，具体以条目为准

这里保存了大量日文近代出版物、古籍、地图和历史档案，适合寻找日文旧书及东亚历史资料。搜索结果会标注公开范围，有些资源只能在日本国立国会图书馆或合作图书馆内查看。

世界各地的珍本与历史档案

Gallica：法国国家图书馆数字资源

  • 访问地址：https://gallica.bnf.fr/
  • 是否注册：浏览和下载公共资源通常不需要
  • 常见格式：PDF、JPEG、TXT、在线阅读

Gallica 收录图书、报纸、手稿、地图、照片、乐谱和录音。法语资料最多，同时也能找到其他语言的历史文献，许多公共版权条目支持下载。

美国国会图书馆数字馆藏

  • 图书入口：https://www.loc.gov/books/
  • 是否注册：不需要
  • 获取方式：PDF、图片、文本或在线阅读

美国国会图书馆开放了大量历史书籍、手稿、地图、照片和档案。它不完全是普通电子书网站，但对历史、文化和地方资料研究非常有价值。下载前可以查看每项资源的 Rights & Access 说明。

Europeana：欧洲文化遗产聚合搜索

  • 访问地址：https://www.europeana.eu/
  • 是否注册：搜索不需要
  • 获取方式：跳转到收藏机构阅读或下载

Europeana 聚合欧洲各地博物馆、图书馆和档案馆的数字资源，包括图书、报纸、艺术品、地图、照片和音频。它更像一个跨机构搜索入口，具体下载格式与授权由原收藏机构决定。

免费有声书

LibriVox：志愿者朗读的公共版权作品

  • 访问地址：https://librivox.org/
  • 是否注册：收听和下载不需要
  • 常见格式：MP3、M4B、在线播放

LibriVox 由志愿者朗读公共版权作品，英文经典文学最多，也包含其他语言。它适合通勤时听书或练习外语听力。由于朗读者来自志愿者社区，不同作品和章节的录音质量可能有所差异。

关于 Z-Library、安娜档案和 Library Genesis

寻找免费电子书时，经常会看到 Z-Library、安娜档案（Anna’s Archive）和 Library Genesis。这类服务通常被称为“影子图书馆”，能够检索大量图书和论文，但其中包括仍受版权保护、未经作者或出版社授权的内容。

本文不提供这些网站的下载入口或绕过限制的方法，原因并不只是版权问题：它们的域名经常变化，搜索引擎和社交平台上存在大量仿冒站点，可能诱导付款、窃取账号或投放恶意文件。在不同国家和地区，下载或传播未获授权的版权作品还可能承担法律责任。

如果一本书仍在正常销售，可以优先尝试：

  1. 查询作者主页或出版社是否提供开放版本；
  2. 在 DOAB、OAPEN、Open Library 或所在地公共图书馆检索；
  3. 使用学校图书馆购买的数据库权限；
  4. 查找正规电子书商店的免费促销、试读或订阅服务；
  5. 购买正版电子书、纸质二手书，或者向图书馆建议采购。

下载前先看懂电子书格式

格式适合场景特点
EPUB手机、平板、Apple Books、Kobo文字会随屏幕自动重排，通常是首选格式
PDF教材、论文、扫描古籍、版式固定的书能保留原版页面，但小屏阅读体验较差
AZW3较老的 Kindle 设备、本地书库Kindle 兼容性较好，功能比 MOBI 完整
MOBI老旧 Kindle 文件已逐渐被 EPUB 和 AZW3 取代
TXT纯文字阅读兼容性好，但通常没有目录、图片和排版
M4B、MP3有声书M4B 通常支持章节和播放进度

需要管理书库、编辑元数据或转换无 DRM 的电子书，可以使用开源软件 Calibre。转换文件时仍应遵守原作品的版权、开放许可证和 DRM 限制。

文件安全与版权检查

下载电子书时，可以用下面几条规则快速排除风险：

  • 正常电子书通常是 EPUB、PDF、AZW3、MOBI、TXT、M4B 或 MP3，不应要求运行 EXE、APK、DMG 或脚本文件；
  • 不要为了下载一本书而关闭系统安全软件或安装来历不明的浏览器扩展；
  • 遇到反复跳转、强制填写银行卡、要求购买“永久会员”或索要账号密码的页面，应立即退出；
  • 优先从作者、出版社、大学、图书馆、政府机构和知名开放项目的官网获取文件；
  • 阅读条目中的 License、Rights、Usage 或 Access Rights，确认是否允许下载、打印、改编和再传播；
  • 公版期限因国家和地区而异，在一个国家进入公共领域的作品，在另一个国家不一定可以自由使用。

写在最后

读书的人也许变少了，但获得知识的渠道从未像今天这样丰富。

可以免费获得一本书，并不意味着我们不需要尊重作者。优先选择公共版权、开放授权和图书馆资源，既能节省费用，也能让阅读这件事更加安心、长久。
