fnm 是什么

fnm（Fast Node Manager） 是一个使用 Rust 编写的 Node.js 版本管理器，支持 macOS、Linux 和 Windows。它可以在同一台电脑上安装多个 Node.js 版本，并根据不同项目的要求快速切换版本。
